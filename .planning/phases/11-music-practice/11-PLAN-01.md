Class-level attributes:

Note on triggers: skill_registry.get_skill_for() matches only the FIRST word of a command (see registry.py line 83). Multi-word triggers like "music start" are never matched by the fast path — they are for router agent visibility only. The functional fast-path triggers are "musik" and "music".

__init__ sets: _session_start (datetime | None = None), _note_count (int = 0), _cents_sum (float = 0.0), _last_note (str = ""), _lock (threading.Lock()).

handle(self, context: dict) -> str | None:
  - Read context.get("cmd", "").strip()
  - If cmd == "_pitch_event": call self._handle_pitch_event(context) and return None
  - Otherwise: call ui_bridge.emit_workspace_change("music") and return the German string "Musikpraxis-Bereich geöffnet."

_handle_pitch_event(self, context: dict) -> None:
  - Read data = context.get("data", {})
  - Under self._lock: if _session_start is None set it to datetime.utcnow(); increment _note_count; add abs(data.get("cents", 0)) to _cents_sum; set _last_note = data.get("note", "")
  - Call self._emit_stats()

_emit_stats(self) -> None:
  - Under self._lock: compute elapsed = (datetime.utcnow() - _session_start).seconds if _session_start else 0; compute avg = _cents_sum / _note_count if _note_count > 0 else 0.0
  - Call ui_bridge.emit_music_session(note_count=self._note_count, avg_cents_deviation=round(avg, 1), time_active_seconds=elapsed, last_note=self._last_note, session_started=self._session_start.isoformat() if self._session_start else "")

Imports: logging, threading, from datetime import datetime, from workspaces.base import Skill, import ui_bridge, from core.logger import get_logger. Use get_logger("music.skill") — NOT logging.getLogger. No print() calls.
