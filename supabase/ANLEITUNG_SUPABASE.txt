SUPABASE — Einrichtungsanleitung
=================================

1. ACCOUNT & PROJEKT
   - supabase.com → "New Project" anlegen
   - Region: Europe (Frankfurt) empfohlen

2. API-KEYS HOLEN
   - Im Dashboard: Settings → API
   - "Project URL"  → SUPABASE_URL
   - "anon public"  → SUPABASE_KEY (für normale Nutzung)
   - "service_role" → nur für Admin-Operationen, geheim halten!

3. KEYS EINTRAGEN
   - Datei: D:\Zuki\persona\config\.env
     SUPABASE_URL=https://xxxx.supabase.co
     SUPABASE_KEY=dein-anon-key

4. TABELLE ANLEGEN
   - Im Dashboard: SQL Editor öffnen
   - Inhalt von D:\Zuki\supabase\migrations\001_init_memory.sql einfügen
   - "Run" klicken → Tabelle "zuki_interactions" wird erstellt

5. ABHÄNGIGKEIT INSTALLIEREN
   pip install supabase

6. VERBINDUNG TESTEN
   from memory import ZukiMemory
   memory = ZukiMemory()
   memory.store_interaction("user", "Test")
   print(memory.get_context(1))
