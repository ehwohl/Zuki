LLM API-KEYS — Einrichtungsanleitung
=====================================

ANTHROPIC (empfohlen für Zuki)
-------------------------------
1. console.anthropic.com → anmelden
2. Links: "API Keys" → "Create Key"
3. Key beginnt mit: sk-ant-...
4. In .env eintragen:
   ANTHROPIC_API_KEY=sk-ant-api03-...

OPENAI (Alternative)
--------------------
1. platform.openai.com → anmelden
2. Oben rechts: API Keys → "Create new secret key"
3. Key beginnt mit: sk-...
4. In .env eintragen:
   OPENAI_API_KEY=sk-...

AKTIVIERUNG
-----------
- Beide Keys in D:\Zuki\workspaces\os\voice\config\.env eintragen
- Anthropic hat Priorität wenn beide gesetzt sind
- Zuki erkennt automatisch ob Key gültig ist:
    Modus : SIMULATION  → Platzhalter, kein Key
    Modus : LIVE        → echter Key erkannt
